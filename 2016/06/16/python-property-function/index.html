Python 中的 property 函数

Jun 16, 2016 | Hits

前几天朋友一起吐槽 Bottle 的代码好神奇, 有好多让人很难一下就看懂的写法. 比如下面这段

Bottle 中environ是个 dict like 的 key/value 变量, 保存了 Request/Response 生命周期中需要的参数. 而 LocalRequest 这个类中 environ 使用了一个 helper function _local_property 来实现多线程工作模式下线程内变量的隔离.

那 _local_property 函数是如何实现这个功能的呢, 这里我来尝试分解一下

首先看一下 property 这个函数的定义以及参数说明

fget is a function for getting an attribute value. fset is a function for setting an attribute value. fdel is a function for deleting an attribute value. And doc creates a docstring for the attribute.

fget 是用来访问属性 fset 用来设置属性, fdel 用来删除属性, 而 doc 则是用来生成 docstring.

按照上面的分析 _local_property 是这样工作的

一句话总结 _local_property 是用来在多线程工作模式中创建替代单线程模式中字典类型的数据类型的函数.
